1. Tap **"+"** FAB to create a new task

   - **None** (uncategorized)

1. Tap on any task in the list
2. The Edit Task screen opens
3. Tap the **"Category"** field

5. Tap **"Save"**

1. Open the **Drawer Menu** (tap Menu icon)
2. In the **Categories** section, you'll see:

3. Tap on any category name
4. The task list filters to show only tasks in that category
5. The drawer closes automatically

### ğŸ’¡ Tips & Tricks

**Tip 1: Default Categories**
- The app comes with 4 default categories
- They cover most common use cases
- Try them before creating new ones

**Tip 2: Empty Categories**
- If a category has 0 tasks, consider deleting it
- Keep your category list clean and relevant

**Tip 3: Color Psychology**
- Colors affect productivity
- Red for urgent (creates urgency)
- Blue for focus (promotes calm)
- Green for wellness (encourages balance)

**Tip 4: Mobile vs. Desktop Thinking**
- On mobile, fewer categories = easier navigation
- Aim for 5-8 active categories max
- More categories = more scrolling

**Tip 5: Consistency**
- Use the same category for similar tasks
- Don't create "Work" and "Work Tasks" - confusing!
- Standardize your naming convention

### âš ï¸ Troubleshooting

**Problem**: Can't delete a category
- **Solution**: Check if it's a default category (Work, Personal, Shopping, Health). Default categories cannot be deleted for data integrity.

**Problem**: Tasks disappeared after deleting category
- **Solution**: Tasks are NOT deleted! They're uncategorized. Filter to "All" or "None" to see them.

**Problem**: Category changes not appearing
- **Solution**: Pull down to refresh the task list. The app uses reactive updates, but a manual refresh can help.

**Problem**: Too many categories, hard to navigate
- **Solution**: Delete unused categories. Aim for 5-8 max. Combine similar categories.
